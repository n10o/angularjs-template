# angularjs-template
angular.js project initial template used gulp.js. Only you need to edit is in "src" directory and bark to "gulp".

# Gulp role
- jade & stylus compile
- js & css minify
- browser hot reload used browserSync

# Requirements
- npm
- bower(npm install -g bower)
- gulp.js(npm install -g gulp)

# Usage
```
npm install
bower install
gulp 
```
